記事を投稿する

このサイトへの記事投稿は、今のところ次の 2 通りで受け付けます。

投稿方法 1（おすすめ）：GitHub から直接記事を書く

GitHub アカウントをお持ちの方は、次のボタンから _posts フォルダに新しい記事ファイルを作成できます。

ファイル名と日付・タイトルなどを自分用に書き換えてから、 そのまま Pull Request を送ってください。

GitHub で新しい記事を書く（テンプレ付き）

  • ファイル名は自動で今日の日付が設定されます。必要に応じて変更してください。
  • title / author / tags / summary も自分用に書き換えてください。
  • 本文はテンプレートの下にそのまま書き足せます。

投稿方法 2：Discussion で記事案を投げる

GitHub に慣れていない方は、次の Discussion カテゴリに 「記事にしたい内容」を書き込んでください。こちらで Markdown 化して取り込みます。

Discussion に記事案を投稿する

書いてもらえると助かる項目：

  • 記事タイトル（仮でOK）
  • ペンネーム
  • 対象バージョン（例：HSP3.6）
  • 対象レベル（入門 / 中級 など）
  • 本文、または自分のブログ記事 URL
